# ★ THE ADVENTURE ★ (冒险)

这是一个基于 [Python](https://www.python.org/) 和 [Textual](https://textual.textualize.io/) 框架开发的现代 TUI (终端用户界面) 文字冒险游戏 (MUD)。

## 🌟 核心特性

- **数据驱动剧本**：游戏内容完全由 JSON 定义，支持复杂的房间跳转、属性变更和条件分支。
- **动态对话系统**：独立的对话引擎，支持一次性剧情和重复性 NPC 交互。
- **沉浸式 UI 反馈**：
    - **打字机效果**：故事文本逐步展现。
    - **状态飘字**：生命、理智等属性变化时在历史记录中实时高亮反馈。
    - **动态氛围控制**：界面边框和背景色会根据玩家的 SAN 值（理智值）或当前状态（如进入对话）动态切换。
- **完善的系统模块**：包含背包系统、人物属性、存档管理、以及丰富的设置选项。

## 🚀 快速开始

### 运行环境
- Python 3.8+
- 安装依赖：`pip install textual rich`

### 启动游戏
```bash
python launcher.py
```

## 🎮 操作说明

| 按键 | 功能 |
| :--- | :--- |
| **1 - 4** | 选择对应的选项 / 菜单项 |
| **P** | 查看人物状态 (Profile) |
| **I** | 打开背包 (Inventory) |
| **D** | 查看日记 (Diary) |
| **S** | 快速保存 (Save) |
| **O** | 系统设置 (Settings) |
| **ESC** | 返回上一级 / 退出当前界面 |

> **提示**：本游戏完全支持鼠标操作，你可以直接点击任何按钮。

## 📂 项目结构

```text
├── launcher.py           # 游戏入口
├── engine/               # 核心引擎层
│   ├── engine.py         # 游戏逻辑与状态管理
│   ├── inventory.py      # 背包管理系统
│   └── effects.py        # 视觉特效组件 (打字机等)
├── view/                 # UI 表现层
│   ├── main_menu.py      # 主菜单
│   ├── game_menu.py      # 游戏主界面 (GamePlayScreen)
│   └── ...               # 其他专项功能界面
└── data/                 # 数据配置层
    ├── rooms.json        # 场景与剧本数据
    └── dialogues.json    # NPC 对话与独白数据
```

## 🛠 开发扩展

- **添加新场景**：在 `data/rooms.json` 中定义新的房间节点。
- **添加新对话**：在 `data/dialogues.json` 中编写 NPC 交互逻辑。
- **条件判定**：支持 `require` (必须满足) 和 `exclude` (满足则隐藏) 字段，轻松实现复杂的任务线。
